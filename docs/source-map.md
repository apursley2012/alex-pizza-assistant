# Source Map: Pizza Ordering Chatbot

## Original Files

- `source/CIS110-Pizza-Ordering-Chatbot-master/CIS110_Pizza_Ordering_Chatbot.py`
- `source/CIS110-Pizza-Ordering-Chatbot-master/CIS110 Pizza Ordering Chatbot.pyproj`
- `source/CIS110-Pizza-Ordering-Chatbot-master/CIS110 Pizza Ordering Chatbot.sln`

## Static Files

- `index.html` contains the browser chatbot interface.
- `css/styles.css` controls the pizzeria/chat layout.
- `js/app.js` contains the browser version of the ordering flow.
- `data/menu.json` stores the menu prices and fee values.

## Logic Preserved

- Name prompt and special creator greeting.
- Size validation.
- Blank toppings and crust validation.
- Numeric quantity validation.
- Carry out/delivery validation.
- Delivery fee.
- Sales tax.
- Coupon threshold message.
- Reset/new order behavior.
